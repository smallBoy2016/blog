javascript 面向对象编程

什么是面向对象程序设计？

对象是一个整体，对外提供一些操作，使用对象时，只关注对象提供的功能，不关注其内部细节。

  • 简称OOP思想，它有三大特征：封装、继承、多态、抽象。
  • OOP的好处：可维护、可重用、可扩展、灵活性高。

1.封装

封装：隐藏内部实现的细节，给外部提供一个接口。

//简单的封装：
function Cat(name,color){
    return{
        name:name;
        color:color     
    }
}
//调用
var cat1 = Cat("大毛","黄色");
var cat2 = Cat("小白","白色");

构造函数模式

构造函数的作用，就是用函数来构造一个对象。对象是有方法和属性的组成

  • 普通的function,就叫函数。
  • 属于一个对象的function ,就叫方法。
  • 用来处理事件的function，就叫事件处理函数
  • 用来构造一个对象的function，就叫做构造函数。

构造函数首字母要大写。如：new Array

//构造函数实例：
function Person(name,sex){
    //属性写在构造函数里，因为每个对象属性各不相同
    this.name = name;
    this.sex = sex; 
}
    //方法写进原型中：所有对象的方法都一样
Person.prototype.showName = function(){
    /*代码块*/ 
}
var person1 = new Person("小白","男");

person1.showName();

this:当前的方法属于谁的。

this啥时候会出问题呢？1.定时器中的this指向window，2.事件。

prototype:（原型 || 模型）

类是于CSS中class一样，给一类元素添加方法和属性。

原型的重要应用-- 可以扩展系统对象的属性和方法。

原型的优先级： 联想行内样式和内嵌样式。

isPrototypeOf()

这个方法用来判断，某个prototype类和实例之间的关系。

alert(Person.prototype.isPrototypeOf(person1));//true

hasOwnProperty()

这个方法用来判断某个属性是属于对象自身的还是来自prototype的。

in运算符

in运输符可用来判断，某个对象实例是否还有某个属性。

alert("name" in Person);//true

in运算符还可以用来遍历某个对象的所有属性。

for(var pro in window){
    document.innerHTML = pro;

}

2.继承

继承：在原来类的基础上，略作修改，成为一个新的类。 父类变了，子类自动跟着改变。

例子：
function Person(name,sex){
    this.name = name;
    this.sex = sex;
}

Person.prototype.showName = function(){
    /*代码块*/
}
//------------------------------
function Worker(name,sex,job){
    //this-->指new出来的Worker对象        
    //也叫够造函数伪装，调用父级的构造函数--为了继承属性
    Person.call(this,name,sex);
    this.job = job;
}
//原型链   通过原型来继承父级的方法
Worker.prototype = Person.prototype;//子类也会影响父类
/*      如何不影响父类？加for循环代码如下：
    for(var i in Person.prototype){
    Worker.prototype[i] = Person.prototype[i];  
}
*/
Worker.prototype.showJob = function(){
    /*代码块*/
};

var W1 = Worker('leo','男','打杂的');

W1.showJob();//打杂的

上面代码案例，就是继承。
