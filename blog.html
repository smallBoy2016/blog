Ajax原理和封装
什么是Ajax?

Asynchronous JavaScript and XML（异步JavaScript和XML)

  • 节省用户操作，时间，提高用户体验，减少数据请求
  • 传输获取数据
Ajax过程详解
	代码案例：
var xhr = new XMLHttpRequest();//1.创建一个ajax对象

xhr.open('get','1.txt',true);
    /*
    2.调用open方法，三个参数：打开方式、地址、是否异步
    */      异步：非阻塞模式，前面的代码不会影响后面代码执行
            同步：阻塞，前面的代码会影响后面代码执行
xhr.send();  //3.提交

//等待服务器返回内容。
/*
    readyState ： ajax工作状态（0，1，2，3，4）
    responseText : ajax请求返回的内容就被存放在这个属性下。
    pn readystate change ： 当readyState改变的时候触发
    status : 服务器状态码（http状态码）
    */
xhr.onreadystatechange = function(){

    if(xhr.readyState == 4){
        if（xhr.status == 200）{
            alert(xhr.responseText);
        }else{
            alert('出错来，Err:'+ xhr.status);
        }           
    }   
}
